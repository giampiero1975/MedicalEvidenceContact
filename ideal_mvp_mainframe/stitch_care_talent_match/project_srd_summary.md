# Software Requirement Document (SRD): Healthcare Matching Platform

## 1. Introduction & Vision
A web platform simplifying the match between socio-sanitary professionals (OSS, nurses, etc.) and healthcare organizations (RSA, clinics, etc.). It features Moodle integration for automatic certificate import and a double-consent privacy system.

## 2. User Roles
- **Professionals (B2C):** Job seekers with profiles, certificates, and calendar management.
- **Business (B2B):** Organizations posting jobs, managing candidates, and scheduling interviews.
- **Admin:** Platform managers with full moderation and oversight capabilities.

## 3. Core MVP Features
- **Authentication:** Specific flows for Professional, Business, and Admin.
- **Profile Management:** Dynamic CVs, certificate imports, and privacy controls.
- **Job Board:** Posting, searching, and filtering job ads.
- **Matching & Interviews:** Candidacy flow, slot-based interview scheduling, and contact exchange upon mutual confirmation.
- **Dashboards:** Role-specific views for activity tracking.

## 4. Key User Stories (Must-Have)
- Professional registration and profile setup.
- Moodle certificate auto-import.
- Business job posting and candidate management.
- Privacy-safe contact exchange post-interview confirmation.
- Admin moderation tools.
